## 👨‍💻 Profile

I am a Software Engineering student at **FPT University Da Nang**, specializing in **Backend Development** with **C# .NET**. Alongside my core backend expertise, I build modern, responsive web applications using **TypeScript** on the frontend, design optimized database structures, and streamline application delivery using **Docker** and automated **CI/CD** pipelines.

I am dedicated to writing clean, maintainable, and high-performance code following industry best practices like SOLID principles and Clean Architecture.
