## 🌟 About Me

Hello! I'm **Phan Hoài An**, a software engineering student at **FPT University Da Nang** 🎓. My primary focus is building robust, high-performance, and scalable backend applications using **C# .NET**, while also having hands-on experience building beautiful, responsive frontends with **TypeScript**.

I love automating workflows, containerizing applications, and maintaining clean code architectures.

- 🏫 **Education:** Software Engineering at **FPT University Da Nang**
- 💻 **Backend Core:** Specialize in **C# / .NET Core**, building RESTful APIs, Clean Architecture, and Entity Framework.
- 🎨 **Frontend Skill:** Building modern and interactive client-side applications using **TypeScript** (React / Next.js).
- 🗄️ **Databases:** Experienced with **Microsoft SQL Server** & **PostgreSQL**.
- 🐳 **DevOps:** Enthusiastic about **Docker** containerization and **CI/CD** pipelines for automated deployments.
